US516153 - Filtrar Por Género

  1. Paso 1:El usuario inicia la acción presionando el botón de filtros, ubicado en la parte superior de la pantalla o en un menú lateral.
  2. Paso 2:El sistema muestra un desplegable con todos los tipos de filtros disponibles, como género, año, calificación, etc. Aquí el usuario selecciona la opción de "Género".
  3. Paso 3:El usuario elige la opción “Género”, indicando que quiere ver solo películas de uno o varios géneros específicos.
  4. Paso 4:El sistema despliega una lista con todos los géneros disponibles. Cada género muestra el número de películas que coinciden, y la lista está ordenada de mayor a menor cantidad de resultados.
  5. Paso 5:El usuario selecciona uno o varios géneros que desea aplicar como filtro. Por ejemplo, puede escoger “Comedia” y “Acción” si quiere ver películas que pertenezcan a ambos tipos.
  6. Paso 6:Una vez que el usuario ha hecho su selección, pulsa el botón “Aplicar” para confirmar su elección.
  7. Paso 7:El sistema cierra automáticamente el menú de filtros, dejando visible solo la lista de películas filtradas.
